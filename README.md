# PostgreSQL_DBA_Tasks

PostgreSQL Learning Lab: An Oracle DBA’s Perspective

A hands-on guide to learning PostgreSQL through the lens of an Oracle DBA. This repository tracks my notes, task-based exercises, and architectural comparisons as I bridge the gap between these two powerful database systems.

**Task1** Install PostgreSQL  
**Task2** Connect to PostgreSQL database  
**Task3** Load a sample database ( for the future tasks)  
**Task4** Understand schema and user/role  
**Task5** Create a tablespace and a table  
**Task6** Backup and restore
